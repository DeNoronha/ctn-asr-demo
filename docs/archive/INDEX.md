## ‚ö†Ô∏è Critical Notes

### Database Status
**The backend database is scheduled down**  
- Grid shows "No records available" when database is offline
- This is normal scheduled behavior
- API endpoint: `https://asr-api.azurewebsites.net/api`
- Check with backend team for uptime schedule

### Kendo React Installation
**Installation is complex with 20+ dependency issues**  
- **MUST read:** [KENDO_INSTALLATION_TROUBLESHOOTING.md](./KENDO_INSTALLATION_TROUBLESHOOTING.md)
- Use exact command provided in guide
- Budget 5 minutes with guide vs 2-3 hours without
- React 19 ‚Üí React 18 downgrade required
